添加邮箱账户

添加单个Outlook邮箱账户到系统

请输入有效的Outlook邮箱地址

从Azure应用程序注册中获取的刷新令牌，用于OAuth2认证

Azure应用程序的客户端标识符

系统会自动识别并使用 IMAP 或 Graph 模式验证账户

可手动指定认证模式；默认 auto 会先尝试 IMAP，再回退 Graph

如果填写，该账户可通过 /web/{邮箱----邮箱密码} 直接打开邮件列表

📋 获取步骤：

  1. 访问 Azure Portal
  2. 注册应用程序并配置权限
  3. 获取Client ID和Refresh Token
  4. 确保应用有邮件读取权限

批量添加邮箱账户

一次性添加多个邮箱账户到系统

该设置会应用到本次批量添加的每一个账户

每行格式（支持两种）：邮箱----密码----刷新令牌----客户端ID 或 邮箱----密码----客户端ID----刷新令牌

📧 推荐购买地址（非广告，只是推荐）： wmemail.com

💡 提示：每行一个账户，使用四个连字符(----)分隔字段

⚠️ 注意事项：

  • 确保所有账户信息格式正确
  • 建议先测试少量账户再批量添加
  • 添加过程中请勿关闭页面
  • 失败的账户会在结果中显示

API接口文档

邮件管理系统的RESTful API接口说明

📡 API基础信息

Base URL

http://localhost:8001

认证方式

需先通过管理员登录页登录，系统使用 refresh token + access token 的 HttpOnly Cookie 鉴权。

响应格式

所有接口返回JSON格式数据，HTTP状态码表示请求结果。

GET /accounts

获取系统中所有已添加的邮箱账户列表，支持分页和模糊搜索。

请求参数

page integer 页码，从1开始，默认: 1
page_size integer 每页数量，范围1-100，默认: 10
email_search string 邮箱账号模糊搜索，可选
tag_search string 标签模糊搜索，可选

响应示例

{ "total_accounts": 25, "page": 1, "page_size": 10, "total_pages": 3, "accounts": [ { "email_id": "example@outlook.com", "client_id": "your-client-id", "auth_mode": "imap", "status": "active", "tags": ["工作", "个人"] } ] }

                                        GET
                                        /emails/{email_id}
                                    

                                    
                                

                                

                                    
获取指定邮箱的邮件列表，支持分页和过滤。


                                    

                                        
路径参数

                                        

                                            

                                                email_id
                                                string
                                                邮箱地址，需要URL编码
                                            

                                        

                                    


                                    

                                        
查询参数

                                        

                                            

                                                folder
                                                string
                                                邮件文件夹 (all, inbox, junk)，默认: all
                                            

                                            

                                                page
                                                integer
                                                页码，从1开始，默认: 1
                                            

                                            

                                                page_size
                                                integer
                                                每页数量，范围1-500，默认: 100
                                            

                                            

                                                refresh
                                                boolean
                                                是否强制刷新缓存，默认: false
                                            

                                        

                                    


                                    

                                        
请求示例

                                        
GET /emails/example%40outlook.com?folder=inbox&page=1&page_size=20&refresh=true

                                    


                                    

                                        
响应示例

                                        
{
  "email_id": "example@outlook.com",
  "folder_view": "inbox",
  "page": 1,
  "page_size": 20,
  "total_emails": 150,
  "emails": [
    {
      "message_id": "INBOX-1",
      "folder": "INBOX",
      "subject": "邮件主题",
      "from_email": "sender@example.com",
      "date": "2024-01-01T12:00:00Z",
      "is_read": false,
      "has_attachments": true,
      "sender_initial": "S"
    }
  ]
}
